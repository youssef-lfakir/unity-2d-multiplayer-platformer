## 🌟 Game Idea & Concept (فكرة المشروع)
This is a **2D local multiplayer platformer game** designed for two players competing or cooperating in a stylized forest environment. 

The core logic of the project revolves around handling distinct control layouts for each player on the same keyboard, alongside robust 2D physics integration to ensure stable and responsive gameplay.

## ✨ Key Features (المميزات الرئيسية)
* **Local Multiplayer System:** Independent script logic allowing dual-player inputs simultaneously (`Player1` and `Player2`).
* **Smart Camera Tracking:** Powered by **Cinemachine** to provide smooth camera follow without manual scripting.
* **Locked Physics Constraints:** Configured `Rigidbody2D` with frozen Z-rotation to prevent sprites from tilting or flipping during movement.
* **Clean 2D Art Integration:** Seamless 2D side-scroller tilesets properly configured with sorting layers for a neat background depth.

## 🛠️ Tech Stack & Tools
* **Game Engine:** Unity 6 (6000.4.3f1)
* **Programming Language:** C#
* **Camera System:** Cinemachine Package
